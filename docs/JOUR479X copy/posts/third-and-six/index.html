plays <- load_cfb_pbp(seasons = c(2019, 2020, 2021, 2022, 2023))
Code 
third_and_six <- plays %>% 
  select(year,pos_team, down, distance, play_type, yards_gained) %>% 
  filter(down == 3, distance == 6)
Code 
play_type_count<-third_and_six %>% 
  group_by(play_type) %>% 
  summarise(count = n())

Of all of the plays run on 3rd and 6, 360 of them were rushing plays. In addition, 20 plays run on 3rd and 6 ended up in rushing touchdowns.

Code 
rushing_plays <- third_and_six %>% 
  filter(down == 3, distance == 6, play_type == "Rush") %>% 
  mutate(mean(yards_gained))

We see that the mean yard average of runs on third down and six on plays that did not end in touchdowns is 5.86.

So far this season, South Florida has run the ball on 3rd and 6 the most doing it 6 times. In the other four recent years, Navy led the nation twice, Army did it once, and Georgia Southern did it once. Navy and Army both run the triple option so it makes a lot of sense that they would be at or near the top of these lists.
